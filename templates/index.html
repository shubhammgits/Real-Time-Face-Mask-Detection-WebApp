About This Project

This project implements a real-time face mask detection system using Deep Learning and Transfer Learning (MobileNetV2). The goal is to classify whether a person is wearing a mask or not from video streams as well as static images. Built with TensorFlow, Keras, and OpenCV.

It's intentionally simple to run no local CUDA/cuDNN or complex setup needed. The model was developed and tested in Google Colab and trained on a combined dataset of multiple Kaggle sources (~18k images) with standard preprocessing and augmentation.
